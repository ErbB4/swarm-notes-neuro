title: ML-Econometric Yield Forecasting Bottleneck
---

**Background:** Yield curve forecasting is a critical financial task where classical econometric models like ARIMA often outperform modern deep learning architectures on daily historical data.

**Question / Future Work:** There is a need to systematically investigate the integration of exogenous macroeconomic signals and the application of uncertainty quantification techniques like conformal prediction to overcome the performance gap between machine learning and traditional econometric models in yield curve tasks. Future work must also address potential biases from frequency-dependent data and rigorous statistical significance testing for model evaluation.

**Why It Matters:** The persistent performance gap suggests that standard deep learning architectures lack necessary inductive biases for financial yield curve data, necessitating a move toward exogenous integration and rigorous probabilistic frameworks.
