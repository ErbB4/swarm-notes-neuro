title: Efficiency in Agentic TSAD
---

**Background:** Univariate time-series anomaly detection systems often rely on multi-agent architectures that require multiple language model calls per data window, creating significant latency and computational bottlenecks.

**Question / Future Work:** Methods are needed to optimize the computational cost and latency of agent-based time-series anomaly detection, particularly through adaptive filtering or selective model execution to reduce the number of required inferences per window.

**Why It Matters:** Reducing the latency of agentic systems is a prerequisite for moving beyond benchmark evaluation to real-time industrial production environments.
