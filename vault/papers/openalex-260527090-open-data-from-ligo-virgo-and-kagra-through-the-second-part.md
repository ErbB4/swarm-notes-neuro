# Open Data from LIGO, Virgo, and KAGRA through the Second Part of the Fourth Observing Run

**Authors**: The LIGO Scientific Collaboration, The LIGO Scientific Collaboration, The LIGO Scientific Collaboration
**Date**: 2026-05-26
**Paper ID**: [openalex:2605.27090](https://arxiv.org/abs/2605.27090)

## Summary

This paper documents the public release of gravitational-wave observational data spanning April 2024 to January 2025, covering the second part of the fourth observing run (O4b) and the ER16 engineering run. The dataset includes calibrated strain time series from the LIGO, Virgo, and KAGRA network, along with auxiliary channels for noise subtraction and detector diagnostics. These contributions facilitate the usage of version 5.0 of the Gravitational-Wave Transient Catalog for further astrophysical research.

## Key Contributions

- Public release of gravitational-wave strain data for the second part of the fourth observing run (O4b) via the GWOSC.
- Integration of engineering run (ER16) data and supplementary detector characterization channels into the public dataset.
- Support for version 5.0 of the Gravitational-Wave Transient Catalog (GWTC-5.0) analysis data products.
