# Inhibiting the right dorsolateral prefrontal cortex selectively enhances unsupervised statistical learning

**Authors**: Pesthy, O., Pesthy, Z. V., Vekony, T., Janacsek, K., Fabo, D., Nemeth, D.
**Date**: 2026-04-01
**Paper ID**: [biorxiv:10.1101/2025.08.08.669288](https://www.biorxiv.org/content/10.1101/2025.08.08.669288)

## Summary

This study investigates the causal role of the dorsolateral prefrontal cortex (DLPFC) in unsupervised statistical learning using inhibitory rTMS. The researchers demonstrate that inhibiting the right, but not the left, DLPFC significantly enhances statistical learning performance. Results indicate that this effect is not mediated by competition with episodic memory but rather by a shift in cognitive processing style towards increased exploratory information-sampling, as evidenced by heightened reaction time variability. These findings refine understanding of prefrontal hemispheric specialization and the mechanisms underlying automatic environmental regularity extraction.

## Key Contributions

- Provides causal evidence that right-lateralized DLPFC activity exerts a suppressive influence on unsupervised statistical learning.
- Demonstrates that inhibiting the right (and bilateral) DLPFC selectively enhances statistical learning performance without affecting episodic memory.
- Proposes that the right DLPFC modulates information processing by shifting the balance between exploitation and exploratory information-sampling strategies.
