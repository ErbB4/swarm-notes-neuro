# Topological Effects in Neural Network Field Theory

**Authors**: Christian Ferko, James Halverson, Vishnu Jejjala, Brandon Robinson
**Date**: 2026-04-02
**Paper ID**: [arxiv:2604.02313](https://arxiv.org/abs/2604.02313)

## Summary

This paper extends neural network field theory to incorporate topological effects by introducing discrete parameters that represent topological quantum numbers within the network architecture. The authors show that this construction can model the Berezinskii--Kosterlitz--Thouless transition and verify key features of the T-duality of the bosonic string. These results bridge the gap between neural network ensembles and quantum field theory, demonstrating that topological physics can be captured through parameter-space statistics.

## Key Contributions

- Formalizes a topological extension of neural network field theory by incorporating discrete parameters representing topological quantum numbers.
- Demonstrates that neural network field theory reproduces the Berezinskii--Kosterlitz--Thouless (BKT) transition, including spin-wave criticality and vortex proliferation.
- Provides a neural network-based verification of the T-duality of the bosonic string, including Buscher rule transformation and current algebra enhancement at the self-dual radius.
