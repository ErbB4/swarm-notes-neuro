# Tiny but Trusted: Efficient Vision-Language Reasoning for Time-Series Anomaly Detection

**Authors**: Xiaona Zhou, Muntasir Wahed, Tianjiao Yu, Constantin Brif, Ismini Lourentzou
**Date**: 2026-05-28
**Paper ID**: [openalex:2605.30344](https://arxiv.org/abs/2605.30344)

## Summary

This paper introduces VisAnomReasoner, a parameter-efficient vision-language model designed to improve interpretability and performance in time-series anomaly detection. To overcome the lack of existing labeled rationales, the authors curate VisAnomBench, which incorporates high-quality natural-language anomaly explanations generated by large models. Experimental results confirm that VisAnomReasoner achieves significant improvements in anomaly localization accuracy and F1 scores, outperforming existing baselines and demonstrating robust generalization to the TSB-AD-U benchmark.

## Key Contributions

- Introduced VisAnomBench, the first time-series anomaly detection benchmark providing natural-language rationales for anomaly intervals.
- Developed VisAnomReasoner, a parameter-efficient VLM that improves precision and F1 scores by >21% on VisAnomBench compared to existing baselines.
- Demonstrated superior cross-benchmark generalization on the TSB-AD-U dataset with double-digit percentage gains in F1 score.
