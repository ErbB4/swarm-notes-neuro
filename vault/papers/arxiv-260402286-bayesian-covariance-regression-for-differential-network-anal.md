# Bayesian covariance regression for differential network analysis of zero-inflated microbiome data

**Authors**: Zichun Xu, Jing Ma
**Date**: 2026-04-02
**Paper ID**: [arxiv:2604.02286](https://arxiv.org/abs/2604.02286)

## Summary

TRECOR is a Bayesian covariance regression framework designed to infer covariate-dependent microbial networks from zero-inflated compositional count data. By modeling latent variables on a phylogenetic tree and decomposing the covariance into a sparse baseline and a low-rank covariate-dependent perturbation, the model captures dynamic network rewiring. The method utilizes a logistic-tree-normal representation to achieve conjugacy, facilitating efficient posterior inference via a Gibbs sampler. Empirical results on simulated and real-world gut microbiome data demonstrate superior performance in identifying differential network associations compared to existing transformation-based approaches.

## Key Contributions

- Introduced TRECOR, a Bayesian covariance regression framework for inferring covariate-dependent microbial networks from zero-inflated compositional counts.
- Proposed a logistic-tree-normal representation that enables full conjugacy and efficient Gibbs sampling by modeling latent variables on phylogenetic tree nodes.
- Demonstrated that the model outperforms standard transformation-based approaches by explicitly accounting for the compositional and zero-inflated nature of microbiome data.
