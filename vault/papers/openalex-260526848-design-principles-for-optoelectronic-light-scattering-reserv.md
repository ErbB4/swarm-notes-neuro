# Design principles for optoelectronic light-scattering reservoir computing at the edge of chaos

**Authors**: G KIM, YongKeun Park
**Date**: 2026-05-26
**Paper ID**: [openalex:2605.26848](https://arxiv.org/abs/2605.26848)

## Summary

This paper establishes systematic design principles for reconfigurable optoelectronic light-scattering reservoir computers by mapping three physical control axes: reservoir dynamics, input coupling, and interconnectivity. The authors identify that memory capacity is maximized at the edge of chaos, evidenced by a near-zero maximal Lyapunov exponent, providing a transferable framework for photonic hardware design. Experimental validation demonstrates that operating at this three-axis optimum enables high-performance chaotic time-series prediction and speech classification.

## Key Contributions

- Established a systematic three-axis design framework for optimizing physical light-scattering reservoirs by linking hardware control variables to computational performance.
- Demonstrated that the peak memory capacity in photonic reservoirs coincides with the edge-of-chaos, characterized by a near-zero maximal Lyapunov exponent.
- Achieved 84.5% classification accuracy on the 10-class Speech Commands benchmark using a physical optoelectronic reservoir operated at the identified hardware optimum.
