# Time-series forecasting with multiphoton quantum states and integrated photonics

**Authors**: Rosario Di Bartolo, Simone Piacentini, Francesco Ceccarelli, Giacomo Corrielli, Roberto Osellame, Valeria Cimini, Fabio Sciarrino
**Date**: 2026-04-17
**Paper ID**: [openalex:2512.02928](https://arxiv.org/abs/2512.02928)

## Summary

This paper presents a quantum reservoir computing framework implemented on a reconfigurable integrated photonic circuit for time-series forecasting. The system encodes input signals into the circuit’s optical phases and employs a feedback mechanism modulated by measured output probabilities. A key finding is that input photon indistinguishability serves as a quantum resource, enabling the system to approximate higher-order nonlinear functions and outperform distinguishable input states in predictive tasks.

## Key Contributions

- Implemented a quantum reservoir computing protocol using reconfigurable integrated photonic circuits and single-photon detection.
- Demonstrated that input photon indistinguishability acts as a resource for approximating higher-order nonlinear functions in time-series forecasting.
- Showed that two-photon indistinguishable states achieve significantly superior forecasting performance compared to distinguishable input states.
