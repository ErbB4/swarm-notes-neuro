# A Shiny micromapST App

**Authors**: Randall Powers, Wendy Martinez
**Date**: 2026-04-14
**Paper ID**: [openalex:2604.12773](https://arxiv.org/abs/2604.12773)

## Summary

This paper introduces a Shiny web application designed to simplify the creation of linked micromaps, a visual alternative to traditional choropleth maps. By providing an intuitive graphical user interface for the existing micromapST R package, the app removes the requirement for complex manual data frame construction. The work serves as a practical tutorial for researchers and analysts looking to visualize statistical summaries connected to spatial areal units more effectively.

## Key Contributions

- Developed a Shiny application providing a graphical user interface for the micromapST R package to simplify visualization of geographically indexed data.
- Streamlined the process of generating linked micromaps, which previously required complex manual data frame specifications.
- Demonstrated the application's utility through tutorials using real-world geographically indexed datasets.
