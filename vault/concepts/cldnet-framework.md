# Conditional Latent Dynamics Network (CLDNet)

> *Auto-generated stub. Edit this file to add more details.*

## Why It Matters

The architecture innovatively decouples spatial grid requirements from hydrodynamic surrogate modeling, a significant shift for physics-informed machine learning on irregular domains.
