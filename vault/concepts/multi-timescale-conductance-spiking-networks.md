# Multi-timescale Conductance Spiking Networks

> *Auto-generated stub. Edit this file to add more details.*

A gradient-trainable spiking neural network framework that achieves rich firing dynamics and high sparsity by parameterizing neuronal conductance timescales.


## Why It Matters
