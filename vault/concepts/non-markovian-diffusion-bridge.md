# Non-Markovian Diffusion Bridge

> *Auto-generated stub. Edit this file to add more details.*

A generative diffusion process where transition dynamics are conditioned on past and future observations, relaxing the standard Markovian assumption.


## Why It Matters

Enables modeling long-range temporal dependencies and path-dependent dynamics by conditioning on arbitrary historical/future states within a diffusion framework.
