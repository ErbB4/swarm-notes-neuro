# Diffusion Forcing

> *Auto-generated stub. Edit this file to add more details.*

A training technique that assigns independent noise levels to individual frames, allowing models to learn joint-trajectory priors instead of traditional frame-to-frame transitions.


## Why It Matters

Central mechanism enabling the framework to learn joint-trajectory priors and avoid error accumulation in dynamical systems.
