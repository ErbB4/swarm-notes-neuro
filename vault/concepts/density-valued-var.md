# Density-Valued VAR

> *Auto-generated stub. Edit this file to add more details.*

A time-series forecasting framework that models the evolution of density functions by mapping mixture weights to a Euclidean space via a generalized logit transform.


## Why It Matters

Provides a principled way to perform forecasting and causal discovery directly on time-evolving probability density functions by mapping them to Euclidean space.
