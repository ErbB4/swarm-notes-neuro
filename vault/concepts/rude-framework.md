# RuDE (Rubric-based Discriminative Evaluation)

> *Auto-generated stub. Edit this file to add more details.*

A framework for predicting a base model's post-training potential by leveraging response discrimination on contrastive pairs constructed from rubric violations.


## Why It Matters

RuDE addresses the challenge of evaluating base model plasticity without performing full post-training, enabling efficient model selection.

## Evidence

> We propose RuDE (Rubric-based Discriminative Evaluation), a unified framework that bypasses the generation gap of base models by leveraging response discrimination.
